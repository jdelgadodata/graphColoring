# graphColoring

`graphColoring` is an R package implementing constructive heuristics, local search methods, metaheuristics and evolutionary algorithms for the graph coloring problem.

## Development status

This repository is currently being prepared. A preliminary version of the package associated with the Master's Thesis **“Implementación de algoritmos de coloración de grafos”** will be published here.

The package is under development and has not yet been submitted to CRAN.
